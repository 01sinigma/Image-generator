# Быстрое исправление: Максимальное использование GPU

## Проблема
GPU используется только на 1% из-за CPU offload.

## Решение: Отключить CPU Offload

CPU offload уже отключен в `config.yaml`. Перезапустите приложение:

```bash
# Остановите текущий процесс (Ctrl+C)
# Затем запустите снова:
python app.py
```

## Что изменилось

1. ✅ **CPU offload отключен** - модель полностью на GPU
2. ✅ **Исправлен параметр torch_dtype** - теперь правильно передается
3. ✅ **GPU будет использоваться на 80-100%** во время генерации

## Ожидаемый результат

После перезапуска:
- **GPU-Util**: 80-100% во время генерации
- **VRAM**: 6-7GB / 8GB
- **Скорость**: 5-15 секунд на изображение (вместо 2+ минут)

## Если появится ошибка "out of memory"

Верните CPU offload:
```yaml
device:
  enable_cpu_offload: true
```

Или уменьшите размер изображений до 512x512.

## Мониторинг GPU

Откройте отдельное окно PowerShell и запустите:
```bash
nvidia-smi -l 1
```

Во время генерации вы увидите реальное использование GPU.
